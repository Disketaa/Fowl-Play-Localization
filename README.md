# Fowl Play

An in-depth mod that adds a bunch of birds. Some of them add ambience, others have useful features and all of 
them try to annoy you as much as they can, just like real birds!

Current mobs:

- Penguin
- Seagull
- Robin
- Pigeon
- Cardinal

Planned additions:

- Blue Jay
- Sparrow
- Grackle
- Crow
- Duck
- Goose
- Falcon
- Other songbirds
